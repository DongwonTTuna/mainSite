# mainSite

Portfolio site built with SvelteKit, Bun, and Cloudflare adapter.

## Setup

```sh
bun install
```

## Developing

```sh
bun run dev
```

## Building

```sh
bun run build
```

Preview the production build with:

```sh
bun run preview
```

## Quality Checks

```sh
bun run check
bun run lint
bun run format:check
bun run test
```
